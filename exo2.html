web development overview

a course that covers the programming, coding and architectural build of websites giving them their functionalities

aim of the course

build web developers who will act as pilots for an impactful build and maintenance of oter industrial sectors

diiferent types of web development

three different types exist which are back end, front end and full stack

back end development

this is the server side dvelopment where the digital structure of the website is built and where API's are linked to the website

front end development

this is more about the user interface build, such that the user experience may be pleasant through easy navigation of the webpages

full stack development

this involves the back and front ends development process
